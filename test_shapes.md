
## Пояснения к реализации:

1. **Тесты для Circle**:
   - Отдельные тесты для проверки создания с положительным радиусом и выброса исключений
   - Параметризованный тест, проверяющий одновременно площадь и периметр для разных радиусов

2. **Тесты для Rectangle**:
   - Отдельные тесты для проверки создания с положительными сторонами
   - Проверка выброса исключений при различных некорректных значениях
   - Параметризованный тест для проверки вычислений

3. **Фикстуры**:
   - Создана фикстура `circle()`, возвращающая круг с радиусом 1
   - Написаны тесты, использующие эту фикстуру для проверки площади и периметра

4. **Особенности**:
   - Все тесты используют `assert` напрямую (без unittest-стиля)
   - Для проверки исключений используется `pytest.raises` с проверкой сообщения об ошибке
   - Результаты вычислений округляются до 5 знаков для корректного сравнения чисел с плавающей точкой
   - Названия тестов читаемые и отражают их назначение

Тесты можно запустить командой:
```bash
pytest test_shapes.py -v
```
