# Тест-кейсы для формы авторизации GitHub

**Объект тестирования:** Страница авторизации GitHub (https://github.com/login)

---

## TC-1: Успешная авторизация с корректными данными

| Поле | Содержание |
|------|-----------|
| **ID** | TC-1 |
| **Название** | Успешная авторизация с корректными данными |
| **Предусловия** | 1. Существующий аккаунт с логином `testuser@example.com` и паролем `ValidPass123`<br>2. Пользователь не авторизован |
| **Шаги** | 1. Открыть страницу https://github.com/login<br>2. В поле "Username or email address" ввести `testuser@example.com`<br>3. В поле "Password" ввести `ValidPass123`<br>4. Нажать кнопку "Sign in" |
| **Ожидаемый результат** | Пользователь успешно авторизован. Произошло перенаправление на главную страницу дашборда GitHub (https://github.com/) |
