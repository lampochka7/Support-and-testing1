

## Как запустить тесты:

1. Убедитесь, что файлы `calculator.py` и `test_calculator.py` находятся в одной директории.

2. Запустите тесты одной из следующих команд:
   ```bash
   python test_calculator.py
   ```
   или
   ```bash
   python -m unittest test_calculator.py
   ```
   или для запуска с подробным выводом:
   ```bash
   python -m unittest -v test_calculator.py
   ```

## Особенности тестов:

1. **`assertEqual`** - используется для проверки равенства значений
2. **`assertAlmostEqual`** - используется для проверки дробных чисел с учетом погрешности
3. **`assertRaises`** - используется для проверки выброса исключения при делении на ноль
4. Каждый тест имеет docstring с описанием того, что он проверяет
5. Тесты покрывают все основные случаи: целые числа, отрицательные числа, дробные числа, граничные случаи

Все тесты должны пройти успешно, если функции в `calculator.py` работают корректно.
